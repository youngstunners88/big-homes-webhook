# Big Homes - SMS Reminder System
# Pre-built SMS templates for local businesses

## Setup
- Twilio: $15/month for phone number
- Or use: Google Voice (free)

## Templates

### 1. Appointment Reminder (24h before)
```
Hi {{CUSTOMER_NAME}}, this is a reminder about your {{SERVICE}} appointment with {{BUSINESS}} tomorrow at {{TIME}}. Reply YES to confirm or call {{PHONE}} to reschedule.
```

### 2. Appointment Reminder (2h before)
```
Your {{SERVICE}} appointment with {{BUSINESS}} is in 2 hours at {{TIME}}. See you soon! Call {{PHONE}} if anything changes.
```

### 3. Post-Service Follow-up
```
Thanks for choosing {{BUSINESS}}! How was your service today? Leave a review: {{REVIEW_LINK}} - It helps us grow!
```

### 4. Missed You Message
```
Hi {{CUSTOMER_NAME}}, we missed you at {{BUSINESS}} today. Would you like to reschedule? Call {{PHONE}} or reply here.
```

### 5. Quote Follow-up
```
Hi {{CUSTOMER_NAME}}, following up on your {{SERVICE}} inquiry. Ready to schedule? Call {{PHONE}} or book online: {{BOOKING_LINK}}
```

### 6. Re-engagement (30 days no contact)
```
Hi {{CUSTOMER_NAME}}, it's been a while! {{BUSINESS}} misses you. Book now and get 10% off your next service: {{BOOKING_LINK}}
```

### 7. Review Request (with link)
```
Thanks for your business, {{CUSTOMER_NAME}}! Would you take 30 seconds to leave a review? {{REVIEW_LINK}} - It really helps small businesses like ours!
```

### 8. Emergency Alert
```
{{BUSINESS}} is currently booking same-day appointments! Need us today? Call {{PHONE}} now - slots filling fast.
```
