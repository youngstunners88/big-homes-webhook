# Big Homes - TinyFish Setup

## What's TinyFish?

Lightweight browser automation via HTTP requests - no heavy SDKs!

**Perfect for:**
- Scraping Google Maps for leads
- Finding businesses without websites
- Research automation

---

## Setup

1. Go to https://tinyfish.ai
2. Sign up (free tier available)
3. Get API key
4. Run:

```bash
export TINYFISH_API_KEY="your-key-here"

# Example: Find plumbers without websites
./tinyfish.sh "https://www.google.com/maps/search/plumbers+in+new+york" "Find 5 plumbers without websites, return their names and phone numbers"
```
